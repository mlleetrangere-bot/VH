🔑 GitHub Personal Access Token (PAT)
Token omogućuje snimanje rezervacija na GitHub repozitorij. Rezervacije se čuvaju u fajlu rezervacije.json na tvom GitHub repozitoriju i dostupne su s bilo kojeg uređaja.

Kako dobiti token:
1. Idi na GitHub.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. Klikni "Generate new token (classic)"
3. Odaberi scope: repo (puna kontrola repozitorija)
4. Kopiraj token i zalijepi ovdje – čuva se lokalno u browseru

⚠️ Token nikad ne dijeli javno ili u chatu!
